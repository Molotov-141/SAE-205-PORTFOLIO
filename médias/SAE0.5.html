SAÉ 1.05 : Traitement de Données

Automatisation et analyse via des scripts Python

Industrialisation des Données

Dans un contexte R&T, savoir automatiser l'extraction et l'analyse de données est crucial. Ce projet visait à transformer des données brutes en informations exploitables pour l'aide à la décision.

Méthodologie Technique

  • Collecte : Requêtage d'API REST via Python (bibliothèque requests).
  • Parsage : Traitement et filtrage de flux de données aux formats JSON et XML.
  • Analyse : Mise en œuvre d'algorithmes de tri et de statistiques descriptives.
  • Restitution : Génération automatisée de rapports au format Markdown.
